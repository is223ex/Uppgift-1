Vad är en regel i CSS och vad består den av?

CSS använder regler för att fastställa hur HTML-kod ska visas på webbplatsen. Regeln kan delas upp i två delar. Selektorer och Deklarationer. En Selektor kan vara till exempel en klass eller en id. Det hänvisar till vad som ska förändras av CSS koden. Deklarationer styr vad för förändring som ska ske på html sidan. Det kan vara färgsättning eller storlek på texten.

Vad är skillnaden mellan id och class? När bör man använda id respektive class?

ID används när koden behöver specifika förändingar i ett element. Det kan till exempel vara att färgen på "det här stycket". Class används när ett flertal element ska användas av exempelvis samma stilsättning. Det används för att den här sidan ska kunna dela upp texten i olika lådor.
